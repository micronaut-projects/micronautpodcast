EPISODE 004

004 - Micronaut Data

Mon, 06 Dec 2021

Sergio del Amo (Micronaut Developer Advocate) talks about Micronaut Data with Graeme Rocher (Micronaut co-founder). Micronaut Data is a database access toolkit that uses Ahead of Time (AoT) compilation to pre-compute queries for repository interfaces that are then executed by a thin, lightweight runtime layer.
